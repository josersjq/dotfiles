# Dotfiles

In pursuit of Configuration Happyness!

## Contents

+ zsh configuration
+ vim configuration
+ tmux configuration
+ git configuration

## Install

1. `git clone https://github.com/josersjq/dotfiles.git ~/.dotfiles`
1. `cd ~/.dotfiiles`
1. `./install.sh`

## Vim Plugins

Vim plugins are managed with [vim-plug](https://github.com/junegunn/vim-plug). To install run `vim +PlugInstall`
